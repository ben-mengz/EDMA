## Steps
1. **Check Parameters**: Verify all essential ptychography parameters are present.
2. **Append Parameter**: If any parameter is missing or needs update, append it.
3. **Save Parameters**: Persist the current parameter set to disk.
4. **Send to Server**: Submit the reconstruction job to the server.
